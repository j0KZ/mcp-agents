================================================================================
                    GITHUB WIKI PUBLISHING INSTRUCTIONS
================================================================================

WHY THE WIKI DOESN'T EXIST YET:
GitHub wikis are separate repositories that don't exist until you:
1. Enable the Wiki feature in settings
2. Create the first page through the web interface

THIS IS A GITHUB REQUIREMENT - YOU MUST DO THIS MANUALLY FIRST!

================================================================================
                              STEP-BY-STEP GUIDE
================================================================================

STEP 1: ENABLE WIKI
-------------------
1. Open browser → https://github.com/j0kz/mcp-agents/settings
2. Scroll to "Features" section
3. Check ✓ the "Wikis" checkbox
4. Click "Save" if needed


STEP 2: CREATE FIRST PAGE (This creates the wiki repo!)
--------------------------------------------------------
1. Go to → https://github.com/j0kz/mcp-agents/wiki
2. Click "Create the first page" button
3. Enter:
   - Title: Home
   - Content: (copy from wiki/Home.md or just type "Home")
4. Click "Save Page"

✅ NOW THE WIKI REPOSITORY EXISTS!


STEP 3: CLONE, COPY, PUSH
--------------------------
Open PowerShell in your project directory and run:

cd D:\Users\j0KZ\Documents\Coding\my-claude-agents

git clone https://github.com/j0kz/mcp-agents.wiki.git wiki-repo

Copy-Item wiki\*.md wiki-repo\ -Force

cd wiki-repo
git add .
git commit -m "Add comprehensive wiki documentation"
git push origin master

cd ..
Remove-Item wiki-repo -Recurse -Force


STEP 4: VERIFY
--------------
Open → https://github.com/j0kz/mcp-agents/wiki

You should see 6 pages:
  ✓ Home
  ✓ Quick-Start
  ✓ Smart-Reviewer
  ✓ Integration-Patterns
  ✓ Troubleshooting
  ✓ README

================================================================================
                           WHAT IF IT DOESN'T WORK?
================================================================================

ERROR: "Repository not found"
→ You haven't enabled Wiki or created first page yet
→ Go back to STEP 1 and STEP 2

ERROR: "destination path already exists"
→ Run: Remove-Item mcp-agents.wiki -Recurse -Force
→ Then try STEP 3 again

ERROR: "Failed to push some refs"
→ Make sure you're in wiki-repo directory
→ Use 'master' branch (wiki uses master, not main)
